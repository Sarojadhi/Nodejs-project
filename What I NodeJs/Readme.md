# What is NodeJs ?

Node.js ke ho?

1. Node.js ek JavaScript runtime ho jun browser bata bahira JavaScript chalaundai cha. Yo mainly server-side programming ko lagi use garincha, jasma JavaScript code server ma run garincha, web apps ko backend banaunda hoina.

2. Node.js Google ko V8 engine use garera JavaScript ko code fast garera chalaucha. Yo event-driven ra non-blocking I/O model use garne gareko cha, jasle garya le bhanda feri fast ra efficient banaucha.

## Node.js ko features:

1. Single-threaded: Ek janale server handle garne garera high performance deko cha.

2. Non-blocking I/O: Server ko kaam garne while request haru wait garne garincha.

3. Cross-platform: Windows, macOS, Linux ma pani kaam garne.

4. Event-driven: Event ko response anusar kaam garne.

5. Rich Ecosystem: npm bata different libraries haru easily install garna milcha.

## Node.js ko upayog:

1. Web server banauna.

2. Real-time apps: Chat applications, live scoreboards.

3. APIs: REST API ya GraphQL API banauna.

4. CLI tools: Command-line tools, automation scripts.

5. Backend for mobile apps.
